RL-based RAN Slicing Optimization for 5G Networks

Published Oct 12, 2023
Source IEEE Access
Relevance 98% Match

Abstract

This paper proposes a deep reinforcement learning (DRL) framework for dynamic resource allocation in O-RAN slicing. By leveraging proximal policy optimization (PPO) agents deployed at the Near-RT RIC, we demonstrate a 24% reduction in latency for URLLC slices while maintaining eMBB throughput. We also introduce a synthetic data augmentation pipeline to handle data sparsity in early deployment phases.
